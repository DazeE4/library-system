SECTION 5: DEPLOYMENT PROCEDURE
═══════════════════════════════════════════════════════════════════════════

STEP 1: PREPARE FILES
  1.1 Review all frontend files
  1.2 Update API endpoints in index.js to production URLs
  1.3 Minify CSS and JavaScript (optional but recommended)
  1.4 Optimize images (if adding custom ones)

STEP 2: UPLOAD TO SERVER
  2.1 Copy index.html to web root
  2.2 Copy index.css to web root
  2.3 Copy index.js to web root
  2.4 Set file permissions: 644 for files, 755 for directories
  2.5 Verify file ownership

STEP 3: CONFIGURE SERVER
  3.1 Enable gzip compression
  3.2 Set cache-control headers:
       - CSS/JS: max-age=2592000 (30 days)
       - HTML: max-age=3600 (1 hour)
       - Images: max-age=31536000 (1 year)
  3.3 Configure CORS headers (if needed)
  3.4 Set security headers:
       - X-UA-Compatible: IE=edge
       - X-Content-Type-Options: nosniff
       - X-Frame-Options: SAMEORIGIN

STEP 4: TEST DEPLOYMENT
  4.1 Open application in browser
  4.2 Test all 5 pages (home, search, mybooks, saved, login)
  4.3 Test responsive design (480px, 768px, 1366px)
  4.4 Test API connectivity
  4.5 Check browser console for errors
  4.6 Test keyboard navigation

STEP 5: MONITORING
  5.1 Monitor error logs
  5.2 Track page load times
  5.3 Monitor API response times
  5.4 Collect user feedback
  5.5 Iterate based on feedback

SECTION 6: CUSTOMIZATION GUIDE
═══════════════════════════════════════════════════════════════════════════

CHANGE LIBRARY NAME:
  Edit index.html:
    <div class="head_name">Your Library Name</div>
    <span class="head_subtitle">Your Subtitle</span>

CHANGE COLORS:
  Edit index.css :root section:
    --primary: #your-color;
    --secondary: #your-color;
    --accent: #your-color;

UPDATE API ENDPOINTS:
  Edit index.js (around line 5):
    const API_BASE = 'https://your-api-server.com/api';

ADD NEW PAGE:
  1. Add <section> to HTML
  2. Add case to showPage() function
  3. Create loadContent() function

MODIFY ANIMATIONS:
  Edit @keyframes in index.css
  Adjust duration and timing functions

SECTION 7: TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Issue: API calls fail (CORS error)
  Solution: 
    • Check API endpoint URL in index.js
    • Verify CORS headers on backend
    • Use browser DevTools Network tab

Issue: Styles not loading
  Solution:
    • Check CSS file is in correct directory
    • Verify file permissions (644)
    • Clear browser cache (Ctrl+F5)
    • Check network tab for 404 errors

Issue: JavaScript not working
  Solution:
    • Check browser console (F12) for errors
    • Verify JavaScript file exists
    • Check file permissions
    • Test in different browser

Issue: Layout broken on mobile
  Solution:
    • Test viewport width with DevTools
    • Check media queries in CSS
    • Verify responsive grid layouts
    • Test on actual mobile devices
