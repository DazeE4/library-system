## 🎓 Developer Guide

### Adding a New Page

1. Add section to HTML:
```html
<section id="newpage" class="hidden">
    <!-- Content -->
</section>
```

2. Add case to showPage() in JavaScript:
```javascript
case 'newpage':
    loadNewPageContent();
    break;
```

3. Create content function:
```javascript
function loadNewPageContent() {
    // Load and display content
}
```

### Customizing Colors

Update CSS variables in :root:
```css
--primary: #your-color;
--primary-dark: #darker-shade;
--primary-light: #lighter-shade;
```

All components automatically update with new colors.

### Adding API Integration

Update fetch URL in JavaScript:
```javascript
const API_BASE = 'https://your-api-server.com/api';

async function callAPI(endpoint, options = {}) {
    const response = await fetch(`${API_BASE}/${endpoint}`, options);
    return await response.json();
}
```
